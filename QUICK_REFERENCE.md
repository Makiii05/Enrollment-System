# Enrollment System - Quick Reference

## Features

-   **User Management** - System users with role-based access control and authentication.
-   **Department Management** - Create and manage academic departments with status tracking.
-   **Course Management** - Manage courses linked to departments with searchable listings.
-   **Subject Management** - Track subjects with lecture/lab hours, units, and type classification.
-   **Curriculum Management** - Create academic curricula and associate subjects.
-   **Prospectus Management** - Manage program prospectuses linked to subjects.
-   **Dashboard Overview** - Display key metrics and statistics on the main dashboard.
-   **Enrollees Chart** - Visual analytics for enrollment trends and data.
-   **Advanced Search** - Search by related model fields with dynamic options.
-   **Custom Theme** - Modern, customized Filament interface with dark primary colors.
-   **Responsive Design** - Mobile-friendly interface that adapts to all screen sizes.
-   **CRUD Operations** - Full Create, Read, Update, Delete functionality for all modules.
-   **Bulk Actions** - Perform actions on multiple records simultaneously.
-   **Form Validation** - Real-time validation and error handling.
-   **Timestamps** - Automatic tracking of created_at and updated_at.

## Tools & Technology Stack

-   **Laravel 12.0** - Modern PHP web application framework.
-   **PHP 8.2+** - Latest PHP version for performance and security.
-   **Filament 4.0** - Powerful admin panel and form builder.
-   **Tailwind CSS 4.1.17** - Utility-first CSS framework for styling.
-   **Eloquent ORM** - Laravel's database query builder and ORM.
-   **Blade** - Laravel's templating engine.
-   **MySQL/SQLite** - Database management with migrations.
